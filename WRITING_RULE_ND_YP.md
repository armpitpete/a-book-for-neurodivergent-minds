# Writing Rule — ND and YP Readers

## Core rule

Always write for neurodivergent readers and young people.

That means the writing must be clear, literal, scannable, and useful when the reader is tired, overloaded, anxious, embarrassed, or short on words.

## Required style

Use:

- plain language
- short sections
- clear headings
- concrete examples
- practical scripts
- one main point per section
- direct wording
- shame-free explanations

## Avoid

Avoid:

- cryptic titles
- clever wording that hides the point
- long theory before help
- dense paragraphs
- moralising language
- too many cross-references
- assuming the reader remembers earlier chapters
- making the reader work hard to find the useful sentence

## Test question

Before treating any chapter or tool as ready, ask:

> Could a tired neurodivergent reader or young person use this without having to decode it?

If the answer is no, simplify it.
