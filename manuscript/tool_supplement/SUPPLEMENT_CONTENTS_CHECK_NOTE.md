## Purpose

This note checks the 12-tool supplement as a sequence.

It is a control note to confirm that every planned tool position now exists and has a clear job.

## Supplement rule

The supplement should not become a second book.

Each tool should be:

- short enough to use on a bad day
- practical rather than reflective-heavy
- plain-worded
- shame-free
- usable without rereading the whole chapter
- built around one clear action
- given in 30-second, 3-minute, and 30-minute versions

## Sequence check

The sequence now moves in a usable order:

1. reduce immediate pressure
2. define a safe finish line
3. find one next step
4. check current capacity
5. check the body floor
6. shrink choices
7. ask for support
8. reduce overload load
9. bridge transitions
10. set a boundary or exit
11. map a social misread
12. hand over important information

This order works because it starts with immediate stabilising tools, then moves into decisions, support, sensory load, transitions, boundaries, repair, and handover.

## Format check

Every planned tool now has the same broad shape:

- status
- what this tool is for
- use this when
- default line
- 30 seconds
- 3 minutes
- 30 minutes
- script or template
- stop point
- linked chapters

Some tools have extra useful sections where appropriate, such as:

- Tool 7: non-speaking note version
- Tool 9: return point
- Tool 10: boring + bounded version
- Tool 11: key words and quick summary
- Tool 12: appointment and next-day handover versions

These extras are useful, but they should not make the tools feel like homework.
