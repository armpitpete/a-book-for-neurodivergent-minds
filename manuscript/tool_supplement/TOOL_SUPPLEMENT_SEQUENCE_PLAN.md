# Tool Supplement Sequence Plan

## Status

Proposed planning note.

This is not yet a final inventory of existing module files.

The supplement should come after the 35-chapter main book.

It should feel like a set of small supports, not extra homework.

## Supplement purpose

The 12-tool supplement should help the reader do one practical thing at a time.

Each tool should be:

- short
- printable or copyable
- usable on a bad day
- plain-worded
- shame-free
- linked to one or more main chapters
- safe to use without reading the whole book again

## Sequence principle

The tools should move from immediate stabilising tools to wider-life tools.

The order should not feel like a school workbook.

The order should feel like:

1. stop the spiral
2. reduce the load
3. make one action possible
4. ask for support
5. repair without shame
6. protect tomorrow

## Tool format rule

Each tool should use the same simple structure:

```md
# Tool X — Title

## What this tool is for

## Use this when

## 30 seconds

## 3 minutes

## 30 minutes

## Script or template

## Stop point

## Linked chapters
```

## Bad-day rule

Every tool must have a 30-second version.

If a tool only works on a good day, it is not finished.

## Good-enough rule

A tool is good enough when a tired reader can use it without:

- understanding the whole book
- making a new system
- proving their distress
- choosing between too many options
- turning the tool into another demand

## What not to do

Do not make the supplement feel like a second book.

Do not make every chapter require a worksheet.

Do not turn tools into moral tests.

Do not make the reader fill in long reflective exercises before getting help.

Do not add clinical language unless it makes the action clearer.
