# Tool 6 — The Choice Shrinker

## What this tool is for

This tool helps you shrink too many choices into a smaller, safer choice.

Too many options can stop action.

A choice can become too big when it feels permanent, public, judged, expensive, risky, or hard to reverse.

The aim is not to choose perfectly.

The aim is to make the choice small enough to enter.

## Use this when

Use this when:

- there are too many options
- you keep comparing choices and cannot move
- every option feels wrong
- the choice feels bigger than it really is
- you are afraid of choosing badly
- a task has turned into a decision trap
- you need a safe-enough choice, not the perfect answer

## Default line

> Can I make this choice smaller?

## 30 seconds

Shrink the choice to two options.

Use this shape:

> Option A: ____.
>
> Option B: ____.

Then ask:

> Which option is safer enough for now?

If two options are still too many, use this:

> Do the smallest reversible step.

Examples:

- open the form, not finish it
- choose one email, not clear the inbox
- pick the easiest food, not the ideal meal
- tidy one area, not plan the whole room
- ask one question, not solve the whole system

## 3 minutes

Use this shape:

1. The choice is: ____.
2. The options are too many because: ____.
3. I can shrink it to: A ____ / B ____.
4. The safer-enough choice for now is: ____.
5. The stop point is: ____.

Example:

1. The choice is: what to do first today.
2. The options are too many because: everything feels urgent.
3. I can shrink it to: A open the appointment letter / B wash the cup.
4. The safer-enough choice for now is: open the appointment letter.
5. The stop point is: find the date and write it down.

## 30 minutes

Use this table only if useful.

| Choice | Too many options? | Shrunk to A/B | Safer-enough choice | Reversible? |
|---|---|---|---|---|
| ____ | yes/no | A: ____ / B: ____ | ____ | yes/no |

If the choice is not reversible, make the next action about information, not commitment.

Examples:

- find the deadline
- ask what happens next
- check the cost
- write a draft but do not send yet
- ask for the options in writing
- sleep before deciding

## Script or template

Use one of these:

- “I need to shrink the choice before I can answer.”
- “Please give me two options, not all the options.”
- “I am choosing a next step, not a whole future.”
- “I need the smallest reversible action.”
- “Good enough is allowed here.”
- “I can choose the safest-enough option for now.”

## Stop point

Stop when the choice has been made smaller.

Do not keep adding new options.

Do not restart the choice because a better answer might exist.

The tool has worked if the next choice is smaller, clearer, or safer than before.
