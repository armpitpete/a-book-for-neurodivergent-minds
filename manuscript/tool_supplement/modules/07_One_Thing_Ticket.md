# One-Thing Ticket

## Status

Expanded draft v0.1.

## Use this when

Use this when the task is too large and your brain cannot find a safe starting point.

This can happen with housework, forms, messages, appointments, work tasks, school tasks, packing, cleaning, or decisions.

## Default line

“One thing only: ____.”

## What this tool does

The One-Thing Ticket gives you permission to do one named thing and stop.

It is not a full plan.

It is a way to cut through too many tasks at once.

## 30 seconds

Write one tiny action:

- open the form
- find the charger
- put the cup by the sink
- write the first line
- send one holding message

Then do only that.

## 3 minutes

Set a timer for three minutes.

Do the named action only.

When the timer ends, you may stop.

Stopping after the ticket is not failure. It means the ticket worked.

## 30 minutes

Make three One-Thing Tickets:

1. one body task
2. one environment task
3. one admin or communication task

Example:

- drink water
- clear the chair
- send one message

Do not add more unless those three are finished and you still have capacity.

## Useful scripts

“I am doing one thing first.”

“I cannot take the whole task at once.”

“One thing is enough to start.”

## Quick summary

- Pick one tiny action.
- Do not turn it into the whole project.
- Stop after the ticket if you need to.
