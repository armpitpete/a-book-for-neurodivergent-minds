# Start here (bad day version)
If you are overloaded, do this in **20 seconds**:

1. Open **[[Tool Index and Script Library (v1)]]**
2. Find the heading that matches what’s happening.
3. Use the **one default line**. Say it once. Repeat once if needed.
4. If it gets unsafe, **leave**. You do not need to “explain it right”.

> Safety note (plain): This book is not medical advice. If you are in immediate danger, call emergency services (UK: 999 / 112). If you need urgent mental health support in the UK, consider Samaritans (116 123) or NHS 111.

# What this book is (scope)
A practical story-manual for neurodivergent people (and the people around them), built for:
- **Misread tone / face**, processing time, and “spaced out” moments
- **Overload → shutdown**, and getting out early without shame
- **Baiting / “banter” traps**, and refusing escalation
- **Repair**, when your words came out wrong but your intent was honest
- **Appointments**, where speed and vagueness destroy clarity

**Pattern (repeatable):**
Start here if… → 30s / 3m / 30m tool → copy/paste scripts → 3‑bullet recap → short mirror scene that makes the tool stick.
